Stage 1: Imports and Linkage Data Preparation (One Time Setup)

This code imports libraries and calculates the **distance matrix** (`linked`) using the data. The distance matrix is the foundation for all hierarchical methods.

# Import necessary libraries for this section
from sklearn.cluster import AgglomerativeClustering
from sklearn.metrics import silhouette_score
from scipy.cluster.hierarchy import linkage, dendrogram
import matplotlib.pyplot as plt

# The linkage function calculates the initial distances (clusters are merged based on this)
# We start with the 'ward' method (minimum variance) for the baseline.
# NOTE: The variable X_scaled should already be defined from the previous K-Means section.
linked = linkage(X_scaled, method='ward')

Stage 2: Dendrogram Generation (Visualizing Merges)

This block plots the **Dendrogram**. Each merge of two clusters is represented by a U-shaped line. The height of the U-line shows the distance at which the clusters were merged. **Use this plot to answer Question 1.**

# STAGE 2: Dendrogram Generation 
# This plot visually represents the hierarchical merging process.
plt.figure(figsize=(12, 8))
dendrogram(linked, orientation='top', distance_sort='descending', show_leaf_counts=True)
plt.title('Dendrogram for Hierarchical Clustering (Current Linkage)')
plt.xlabel('Sample Index')
plt.ylabel('Distance')
plt.show()

Stage 3: Agglomerative Clustering (Model Fitting)

This is where the model is actually created. `AgglomerativeClustering` takes the linkage logic (calculated in Stage 1) and **cuts the hierarchy** to create the final $k=4$ clusters. The result is stored in `h_labels`.

# STAGE 3: Agglomerative Clustering 
# We explicitly set n_clusters=4 and define the linkage method.
h_cluster = AgglomerativeClustering(n_clusters=4, linkage='ward')
h_labels = h_cluster.fit_predict(X_scaled)

Stage 4: Visualization and Score Calculation (Result)

This final block calculates the **Silhouette Score** (measuring cluster separation and cohesion) and plots the resulting clusters. **Record the printed Silhouette Score in the table.**

# STAGE 4: Visualization and Score Calculation
score = silhouette_score(X_scaled, h_labels)

plt.figure(figsize=(8, 6))
plt.scatter(X_scaled[:, 0], X_scaled[:, 1], c=h_labels, cmap='viridis', marker='o')
plt.title(f"Hierarchical Clustering (Linkage: Ward, k=4)")
plt.xlabel("Scaled Global Sales") # Example feature label
plt.ylabel("Scaled North American Sales") # Example feature label
plt.colorbar(label='Cluster Label')
plt.show()

print(f"Silhouette Score: {score:.4f}")
